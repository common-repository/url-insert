This plugin helps you to insert and display your actual blog url.

== Description ==

*	Replaces the Blog-URL with [uri]
*	Inserts the Blog-URL instead of [uri]

== Installation ==

This section describes how to install the plugin and get it working.

1. Copy folder with file(s) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

For help visit http://wewa.kilu.de/2008/03/01/plugin-insert-url/
